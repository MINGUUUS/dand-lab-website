# DAN-D Lab GitHub Pages Site

This folder contains a static GitHub Pages-ready homepage for DAN-D Lab.

## Files

- `index.html`: full one-page lab website
- `styles.css`: responsive styling
- `script.js`: mobile navigation and publication filters
- `assets/hero-ai-nanomaterials.png`: temporary generated hero image

## Recommended GitHub Pages Setup

1. Create a new GitHub repository, for example `dand-lab-knu.github.io` or `dand-lab-website`.
2. Upload all files in this folder to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, select `Deploy from a branch`.
5. Select the `main` branch and `/root`.
6. Save and wait for the Pages URL to appear.

## Before Public Launch

Update these placeholders in `index.html`:

- Office
- Email
- Any official lab location details
- Profile links such as Google Scholar, LinkedIn, GitHub, and X
- Real lab photos or professor profile photo
